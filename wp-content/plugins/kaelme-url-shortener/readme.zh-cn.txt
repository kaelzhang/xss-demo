=== kael.me URL shortener plugin for wordpress ===
标签: twitter, tweet, shorten, url, short-url, url-shortener
至少需要: 2.7
最高兼容: 2.8.5

== Description ==

kael.me URL shortener可以让您自定义形如http://yoursite/-abc的网址来访问原来的链接，并且轻松的管理这些短网址，修改它们，并且查看点击次数.

同时支持301永久转向和307临时转向。


== 安装 ==

下载本插件，并解压，将'km-url-shortener'文件夹上传到您的wordpress的'/wp-content/plugins/' 目录, 然后激活插件，然后接下来，就可以添加您自己的短网址了。

这样就完成了！


== 使用说明 ==

如果您的服务器不支持rewrite_module, 您需要在kael.me URL shortener插件页面进行简单的设置， 将'Oops, my rewrite_module is disabled!'选择框选中，并且保存设置.
